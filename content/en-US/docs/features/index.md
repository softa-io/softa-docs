# Features

Softa is built around a unified metadata model and a production-first foundation.

## Highlights

- **Metadata**: Define entities/fields/UI as versioned metadata.
- **Flow**: Automation, approvals, scheduled jobs, and AI integration.
- **Security**: ABAC with row/field-level control, encryption & masking.
- **Integration**: OpenAPI-first and configurable integrations.
- **Timeline**: Query effective data by date; track business events.
- **I18n**: Localize UI/messages/data for global teams.
