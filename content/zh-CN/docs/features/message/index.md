# Message Starter

Message Starter 为 Softa 应用提供统一的消息能力：

- **邮件**：发送、接收邮件，以及渲染邮件模板
- **短信**：发送短信、批量发送、模板渲染与失败重试
- **收件箱**：推送应用内通知，并为用户创建可处理的待办项

## 依赖

```xml
<dependency>
  <groupId>io.softa</groupId>
  <artifactId>message-starter</artifactId>
  <version>${softa.version}</version>
</dependency>
```
