# 特性

Softa 围绕统一的元数据模型构建，并提供面向生产的企业级底座能力。

## 重点能力

- **元数据**：实体/字段/UI 统一抽象，支持版本化与多环境发布。
- **Flow**：自动化、审批、定时任务与 AI 集成。
- **安全**：ABAC，行级/字段级权限控制，加密与脱敏。
- **集成**：OpenAPI 优先，可配置化集成与编排。
- **时间轴**：按生效日期查询有效数据，追踪业务事件。
- **多语言**：界面/消息/数据多语言，面向全球团队。
