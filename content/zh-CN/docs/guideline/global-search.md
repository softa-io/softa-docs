## 打开搜索面板

| 入口                     | 操作方式                                       |
| ------------------------ | ---------------------------------------------- |
| 应用内任意位置           | 按 **⌘ K**（macOS）/ **Ctrl K**（Windows / Linux） |
| 顶部搜索框（桌面端）   | 点击页面顶部的搜索框                           |
| 顶部搜索图标（移动端） | 点击右上角的放大镜图标                         |

三种方式访问的是同一套数据——任选最顺手的一种即可。

## 通过输入搜索

输入几个字符即可。面板会在 **页面名称、模块名称，以及（当前活动类别下）员工或部门名称** 中进行匹配。

```
employees       会匹配「员工」页、「员工文档」等
salary          会匹配名称或描述中包含「salary」的任意页面
```

## 按类别筛选 — 前缀语法

在输入开头加上特殊字符，可以把搜索限定在单一类别中：

| 前缀 | 搜索对象           | 示例            |
| ---- | ------------------ | --------------- |
| `/`  | 页面与功能         | `/leave`        |
| `@`  | 员工               | `@Alice Wang`   |
| `#`  | 部门               | `#Engineering`  |
| `?`  | 帮助文章（即将推出） | `?onboarding`   |

启用前缀后，输入框前会出现一个小的 **类别标签**（例如「员工」）。前缀字符会转化为该标签，
因此输入区保持简洁，只显示你真正输入的关键词。

## 清除或切换类别

- 在 **输入为空** 时按 **退格（Backspace）** → 去掉标签，回到全局搜索
- 点击标签上的 **×** → 同上
- 点击输入框 **右侧的 ×** → 仅清空已输入文字（标签保留）

## 最近使用

打开面板且尚未输入任何内容时，上方会显示 **最近使用** 列表——列出你最近访问过的页面与记录。排序会综合两类信号：

- **最近使用时间** — 越新的越靠前
- **使用频次** — 越常打开的越靠前

因此常用项即使几天未用，也往往会留在较前位置；偶尔才访问一次的内容则会随时间自然下沉。

⚠️ 「最近使用」数据 **仅保存在当前设备 / 浏览器**。更换设备或清除浏览器数据会重新开始一份列表；不会同步到云端，其他用户也看不到。

## 面板内的快捷键

| 按键                 | 作用                       |
| -------------------- | -------------------------- |
| **↑ / ↓**            | 在上 / 下移动高亮项        |
| **⏎（回车）**        | 打开当前高亮项             |
| **Esc**              | 关闭面板                   |
| **⌘ K** / **Ctrl K** | 在任意位置打开 / 关闭面板 |

## 找不到想要的内容？

- 检查关键词是否拼写错误
- 尝试 **不加前缀** 的普通搜索——目标可能在其他类别里
- `?` 帮助文章尚未上线——如需产品说明，请联系 HR / IT
- 仍然没有？可能是你没有访问该功能的 **权限**。请向管理员申请权限后再试。
